# Image Segmentation for Object and Region Identification

## Overview

This project implements a deep learning-based semantic image segmentation system using U-Net architecture with MobileNetV2 transfer learning. The model performs pixel-level object segmentation and supports real-time visualization through a Streamlit dashboard.

## Features

- Semantic Image Segmentation
- U-Net + MobileNetV2 Architecture
- Real-Time Prediction
- Probability Heatmaps
- Segmentation Overlay Visualization
- Streamlit Dashboard
- Webcam-Based Inference

## Technologies Used

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Streamlit
- Matplotlib

## Dataset

Oxford-IIIT Pet Dataset
